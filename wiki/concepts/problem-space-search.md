# Problem Space Search

## Overview
A framework from Newell and Simon's 1950s cognitive science research modeling problem-solving as search through a graph of possible states. It treats cognition as navigation — finding a path from an initial state to a goal state through a space of intermediate possibilities.

## Across Episodes
- **S2E5**: Tree of Thoughts explicitly revives this framework, grounding LLM planning in the same conceptual model Newell and Simon used for human symbolic problem-solving decades before neural networks existed. → episodes/S2E5.md
