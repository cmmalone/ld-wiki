# Inter-Agent Misalignment

## Overview
Inter-agent misalignment refers to failures that arise specifically in multi-agent systems when coordination between agents breaks down. Examples include handoff parsing errors (one agent produces output another cannot interpret), contradictory conclusions with no resolution mechanism, and sub-agents drifting in different directions due to ambiguous orchestrator instructions.
