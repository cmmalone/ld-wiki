# Agentic Loop (Observe–Reason–Act)

## Overview
The agentic loop is the core repeating cycle that distinguishes an AI agent from a one-shot chatbot: the system perceives its environment (observe), decides what to do (reason), takes a consequential action (act), and then repeats. Each iteration can update the agent's understanding and change the state of the world. This loop is the structural backbone of virtually all agent architectures discussed in the AI agent literature.
