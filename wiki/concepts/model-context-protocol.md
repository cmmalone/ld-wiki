# Model Context Protocol (MCP)

## Overview
MCP is an open standard announced by Anthropic in November 2024 for connecting AI agents to external tools and data sources — calendars, email, databases, code repositories, and more. It replaces a fragmented ecosystem of bespoke point-to-point integrations with a single universal protocol, analogous to USB-C as a universal hardware connector. OpenAI and Google DeepMind have both adopted it, making it a de facto industry standard.
