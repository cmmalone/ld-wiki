# Agent Memory Taxonomy (Semantic / Episodic / Procedural)

## Overview
A framework for classifying the kinds of information different agent memory systems store, borrowed loosely from cognitive science. Semantic memory is factual world knowledge (the domain of RAG). Episodic memory is the record of task events and observations (the domain of MemGPT). Procedural memory is know-how — how to perform actions — which agent skill systems externalize and load on demand.
