# Memory vs. Planning in AI Agents

## Overview
A conceptual distinction between two different agent capabilities: memory (what information an agent can access or retain, e.g. context window contents) and planning (whether an agent can model paths forward, consider alternatives, and adjust strategy mid-task). They are often conflated but address different failure modes.
