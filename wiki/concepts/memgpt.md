# MemGPT

## Overview
A memory architecture for LLM agents, proposed by UC Berkeley researchers in late 2023, that applies operating system virtual memory principles to language models. The agent treats its context window as RAM and external storage as disk, actively paging information in and out via tool calls to maintain coherence across tasks longer than any fixed context window can hold.
