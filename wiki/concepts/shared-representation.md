# Shared Representation Problem

## Overview
In multi-agent communication, a shared representation is a common framework — vocabulary, coordinate system, or structural convention — that allows agents to describe the same underlying reality in ways the other can interpret. Without it, agents talk past each other, consuming communication budget without making progress on the task.
