# Virtual Memory and Paging

## Overview
An operating system technique, developed decades ago, that creates the illusion of a larger memory space than physically exists in RAM. The OS transparently moves 'hot' (frequently needed) data into fast RAM and pages 'cold' data out to slower disk storage. Programs interact with a seamless virtual address space and do not manage this themselves.
