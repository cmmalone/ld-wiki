# Multi-Hop Reasoning

## Overview
Multi-hop reasoning refers to answering questions that require retrieving and chaining together multiple pieces of evidence — each 'hop' is a fact that points toward the next relevant fact, until a final answer can be derived. It is a standard benchmark class in NLP used to test whether a system can integrate information across sources rather than pattern-match to a single retrieved passage. HotpotQA is a well-known multi-hop QA dataset.
