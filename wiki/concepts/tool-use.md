# Tool Use (LLMs)

## Overview
Tool use refers to an LLM's ability to invoke external capabilities — web search, APIs, form submission, calendar access — beyond generating text. It is what allows a model to reach outside itself and affect the world. Tool use is a necessary but not sufficient condition for agency: a model that searches once and stops is using a tool but not running an agent loop.
