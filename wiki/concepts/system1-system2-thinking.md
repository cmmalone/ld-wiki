# System 1 and System 2 Thinking

## Overview
A model of human cognition from Daniel Kahneman's book Thinking Fast and Slow. System 1 is fast, automatic, and associative; System 2 is slow, deliberate, and effortful. System 2 is better suited to hard problems requiring careful reasoning but is more costly to engage.

## Across Episodes
- **S2E5**: Used in S2E5 as an analogy: raw LLM token generation approximates System 1, chain-of-thought moves partway toward System 2, and Tree of Thoughts represents a genuine attempt to impose System 2 reasoning on top of an LLM. → episodes/S2E5.md
