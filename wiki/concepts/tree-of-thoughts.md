# Tree of Thoughts

## Overview
A 2023 LLM prompting and planning framework that structures reasoning as a tree search rather than a linear chain. At each step, the model generates multiple candidate next steps (branches), evaluates their promise, prunes dead ends, and can backtrack to earlier nodes — enabling deliberate planning instead of reactive generation.
