# Agent Scaffolding (Harness / Orchestration Layer)

## Overview
Agent scaffolding refers to the architecture surrounding an LLM — tools, memory management, orchestration logic, and system integrations — that determines how the model interacts with the world. Scaffolding is distinct from the model itself and can have a dramatic effect on real-world performance.

## Across Episodes
- **S2E6**: S2E6 notes that research shows the same LLM can exhibit up to a six-fold performance difference depending on the scaffolding around it, making it misleading to interpret benchmark scores as a measure of raw model capability alone. → episodes/S2E6.md
