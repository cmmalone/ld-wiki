# Context Window

## Overview
The finite block of tokens — typically including instructions, conversation history, documents, and tool outputs — that a language model can 'see' at any given moment when generating a response. Everything outside this window is functionally invisible to the model. Measured in tokens, where one word is roughly 0.75 tokens.
