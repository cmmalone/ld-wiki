# Relay Inference

## Overview
Relay inference is a multi-agent communication strategy in which a stronger agent speaks first, establishing the representational and communicative conventions for the task, while a weaker agent follows that lead rather than negotiating terms jointly. It is a leader-follower pattern rather than symmetric collaboration.
