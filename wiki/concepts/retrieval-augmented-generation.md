# Retrieval Augmented Generation (RAG)

## Overview
A pattern in which an external system retrieves relevant documents or facts from a knowledge store and injects them into the model's context before the model generates a response. The model is a passive recipient of this assembled context rather than an active participant in the retrieval process.

## Across Episodes
- **S2E4**: S2E4 contrasts RAG with MemGPT: RAG is retrieval done *to* the model by an external orchestrator, while MemGPT has the agent itself deciding when and what to retrieve — a distinction framed as passive vs. active memory management. → episodes/S2E4.md
