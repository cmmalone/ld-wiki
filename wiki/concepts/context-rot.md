# Context Rot

## Overview
A failure mode in long agent sessions where repeated compaction — summarizing summaries — causes incremental detail loss that accumulates over time. Symptoms include the agent forgetting earlier decisions, asking questions already answered, or making suggestions that contradict its earlier reasoning.
