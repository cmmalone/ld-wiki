# Toolformer

## Overview
Toolformer is a Meta AI research approach (early 2023) that trains a language model to decide autonomously when to invoke external tools such as a calculator, search engine, or calendar. Rather than being prompted with instructions, the model is fine-tuned on a dataset of API calls filtered for ones that genuinely improved next-token prediction. The result is a model with internalized tool-selection intuitions.

## Across Episodes
