# Tool Use in Large Language Models

## Overview
Tool use refers to a language model's ability to call external systems — search engines, calculators, calendars, databases, code interpreters — rather than relying solely on knowledge encoded during training. It is the mechanism that lets an agent take actions with real-world consequences and ground its reasoning in current, verifiable information. Tool use is widely considered the capability that transforms a text generator into an agent.
