# Task Parallelizability

## Overview
Task parallelizability describes how readily a problem can be decomposed into independent subtasks that can be worked on simultaneously without one depending on the output of another. Highly parallelizable tasks are natural fits for multi-agent systems; tasks with strong sequential dependencies are not.
