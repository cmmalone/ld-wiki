# ReAct Loop (Observe-Reason-Act)

## Overview
The ReAct loop is a pattern for LLM-based agents in which the model alternates between reasoning about its situation and taking actions, then observing the results before reasoning again. Originally introduced in the ReAct paper, it formalizes the idea that agency requires a feedback cycle — not just a sequence of actions. The loop continues until the agent determines its goal is met.
